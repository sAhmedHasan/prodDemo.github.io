About This Project

This project showcases a modern product landing page for a fictional skincare serum. The goal was to create an interactive and visually engaging web experience using a combination of 3D design tools and front-end technologies.

Tools Used

  • Blender: 3D Modelling and Texturing
  • Spline: 3D Scene Creation, Lighting, Animations
  • HTML, CSS, JS: Web Development
